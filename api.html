API reference¶

Api¶

Interact with various remote sensing data query and download APIs

api.remote.UsgsApi()
api.remote.EspaApi()
api.remote.ScihubApi()

Indexing¶

Register datasets to the datacube database.

indexing.add_product(description, name) Add a new product to the database given a product description dictionary
indexing.add_product_from_yaml(description, name) Wrapper to add a product directly from a yaml file
indexing.add_product_from_recipe(recipe[, name]) Wrapper to add a product from a recipe name
indexing.add_dataset(pr, dt, metadict, file) Add a dataset to the datacube database
indexing.wkt_to_proj4(wkt) Utility to convert CRS WKT to CRS in proj4 format
indexing.metadict_from_netcdf(file, …[, …]) Get metadata dictionary for netcdf dataset written using write_dataset_to_netcdf

Ingestion¶

Generate metadata to prepare indexing/ingestion of datasets

ingestion.bioclimatics.metadata_convert(path) Prepare metadata prior to datacube indexing
ingestion.landsat_espa.metadata_convert(path) Prepare metatdata prior to datacube indexing
ingestion.srtm_cgiar.metadata_convert(path, …) Prepare metadata prior to datacube indexing
ingestion.s2_l2a_20m.metadata_convert(path) Prepare metatdata prior to datacube indexing

I/O¶

Read and write data

io.vector_db.from_geobox classmethod(function) -> method
io.vector_db.VectorDb() Query, read and write geometries between python’s memory and the database
io.vector_db.VectorDb.load_training_from_dataset(…) Retieves training data from the database based on intersection with an xr Dataset
io.vector_db.load_segmentation_from_dataset(…) Retrieve a segmentation intersecting with a geoarray from the database

Land cover change (lcc)¶

Interface to perform Land cover change detection and classication

Base class¶

lcc.bitemporal.BaseBiChange(array, affine, crs) Parent class implementing generic methods related to change detection results handling.
lcc.bitemporal.BaseBiChange.from_geoarray(…) Instantiate class from a geoarray (xarray read with datacube.load)
lcc.bitemporal.BaseBiChange.run(other) Run change detection using algorithm defined in _run
lcc.bitemporal.BaseBiChange.filter_mmu(min_area) Filter clumps of pixels smaller than min_area
lcc.bitemporal.BaseBiChange.label_change(…) Label change array
lcc.bitemporal.BaseBiChange.filter_no_change(fc) Filter labeled change polygons that have the same before and after label
lcc.bitemporal.BaseBiChange.to_db(fc, meta, …) Write feature collection returned by label_change to the antares3 database
lcc.bitemporal.BaseBiChange.read_land_cover(name) Read the specified land cover map covering the extent of the instance array

Implemented algorithms¶

lcc.bitemporal.distance.BiChange(array, …) Antares implementation of a simple distance based bi-temporal change detection algorithm
lcc.bitemporal.imadmaf.BiChange(array, …) Antares implementation of iMAD-MAF change detection algorithm

Transform module¶

A standardized interface to perform transformation of single arrays or between two arrays

Base classes¶

lcc.transform.TransformBase(X) Metaclass to support single array transform.
lcc.transform.TransformBase.transform() Run transformation defined by children instance
lcc.transform.BitransformBase(X, Y) Metaclasss to support two array transform
lcc.transform.BitransformBase.transform() Run transformation defined by children instance

Implemented transformation method¶

lcc.transform.elliptic.Transform(X[, …]) Antares implementation of elliptic envelop thresholding transformation
lcc.transform.irmad.Transform(X, Y[, …]) Antares implementation of the IR-MAD transformation
lcc.transform.kapur.Transform(X[, band, …]) Antares implementation of Kapur, Sahoo and Wong entropy based array thresholding
lcc.transform.mad.Transform(X, Y[, lmbda, …]) Antares implementation of the MAD transformation of two array
lcc.transform.maf.Transform(X[, shift]) Antares implementation of MAF transformation

Modeling¶

Interface to various statistical models running on numpy arrays

Base class¶

modeling.BaseModel([categorical_features]) This class works as a wrapper to have a single interface to several models and machine learning packages.
modeling.BaseModel.fit(X, y) This method will train the classifier with given data.
modeling.BaseModel.predict(X) When the model is created, this method lets the user predict on unseen data.
modeling.BaseModel.load(filepath) Read object from file
modeling.BaseModel.save(filepath) Write entire object to file
modeling.BaseModel.from_db(name) Instantiate an object from a children class of BaseModel reading it from the database
modeling.BaseModel.to_db(name[, recipe, …]) Write the instance of the class to the datbase
modeling.BaseModel.hot_encode_training(X) Apply one hot encoding to one or several predictors determined by the list of indices of the hot_encode attribute
modeling.BaseModel.hot_encode_predict(X) Hot Encode data on which prediction is to be performed
modeling.BaseModel.remove_outliers(X, y[, …]) Performs outliers detection and removal using Isolation Forest anomaly score

Implemented models¶

modeling.supervised.rf.Model([…]) Antares implementation of scikit learn random forest classifier
modeling.supervised.xgb.Model([…]) Antares implementation of Gradient Boost classifier
modeling.supervised.lgb.Model([…]) Antares implementation of Microsoft’s Light Boost classifier

Overlay¶

Overlay operations: conversions (vector to raster to vector) and extractions

overlay.conversions.rasterize_xarray(fc, dataset) Rasterize a feature collection using an xarray dataset as target
overlay.conversions.train_object_to_feature(x) Convert a Django object, part of a QuerySet, to a geojson like feature
overlay.conversions.predict_object_to_feature(x) Convert a PredictObject to a feature
overlay.extractions.calculate_zonal_statistics(…) Receives an array with labels and indexes for those labels.
overlay.extractions.zonal_stats_xarray(…) Perform extraction and grouping using pandas’ groupby method

Segmentation¶

Interface to various segmentation algorithms operating on numpy arrays and xarray ```Dataset``s

Base class¶

segmentation.BaseSegmentation(array, affine, crs) Parent class implementing generic methods related to running spatial segmentation algorithms on raster data, converting input and output data and interacting with the database.
segmentation.BaseSegmentation.from_geoarray(…) Instantiate class from a geoarray (xarray read with datacube.load)
segmentation.BaseSegmentation.segment() Run segmentation
segmentation.BaseSegmentation.polygonize([…]) Transform the raster result of a segmentation to a feature collection
segmentation.BaseSegmentation.to_db(meta_object) Write the result of a segmentation to the database

Implemented algorithms¶

segmentation.slic.Segmentation(array, …[, …]) Antares implementation of scikit-image’s SLIC segmentation algorithm
segmentation.bis.Segmentation(array, affine, crs) Antares implementation of Berkeley Image segmentation algorithm

Validation¶

Utilities to compute validation metrics on existing results

validation.prepare_validation(fc_valid, fc_test) Generate area weighted confusion matrix
validation.validate(y_true, y_pred[, …]) Compute user’s and producer’s accuracy for each class and overall accuracy
validation.query_validation_intersect(…[, …]) Query intersecting records from the validation and the predictClassification database table
validation.pprint_val_dict(d) Prints the dictionary returned by madmex.validation.validate as a nicely formated table

Wrappers¶

Wrapper functions to be called in scripts and command lines

wrappers.predict_pixel_tile(tile, model_id) Run a model in prediction mode and generates a raster file written to disk
wrappers.extract_tile_db(tile, sp, …) Function to extract data under training geometries for a given tile
wrappers.gwf_query(product[, lat, long, …]) Run a spatial query on a datacube product using either coordinates or a region name
wrappers.segment(tile, algorithm, …) Run a segmentation algorithm on tile
wrappers.predict_object(tile, model_name, …) Run a trained classifier in prediction mode on all objects intersection with a tile
