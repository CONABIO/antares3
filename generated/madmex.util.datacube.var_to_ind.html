madmex.util.datacube.var_to_ind¶

madmex.util.datacube.var_to_ind(variables, product)[source]¶

Helper to get a list of index from a list of variables names

Useful for telling the modeling module which are the categorical variables that should be encoded using One Hot Encoding

Parameters:
  • names (list) – List of strings corresponding to existing variable names in the product
  • product (str) – Name of an existing datacube product
Returns:

LIst of integer corresponding to dataset positions
Return type:

list

Example

>>> from madmex.util.datacube import var_to_ind

>>> ind = var_to_ind(['blue', 'red', 'swir2'], 'ls8_espa_mexico')
>>> print(ind)
