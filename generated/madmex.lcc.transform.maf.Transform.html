madmex.lcc.transform.maf.Transform¶

class madmex.lcc.transform.maf.Transform(X, shift=(1, 1))[source]¶

Antares implementation of MAF transformation

This transform maximizes the autocorrelation for the image. The bands are order by autocorrelation with the first band having the maximum autocorrelation and subjected to be uncorrelated with the other bands. This process is useful in the sense that it improves the spatial coherence of the IR-MAD transform.
