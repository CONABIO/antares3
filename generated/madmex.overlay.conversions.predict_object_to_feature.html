madmex.overlay.conversions.predict_object_to_feature¶

madmex.overlay.conversions.predict_object_to_feature(x, crs=None)[source]¶

Convert a PredictObject to a feature

This function is often meant to be called in a list comprehension whose iterator is a django QuerySet. The feature has a single attribute corresponding to the database object id

Parameters:
  • x (PredictObject) – Object extracted from the database
  • crs (str) – proj4 string to reproject to. Can be extrated from a geoarray using geoarray.crs._crs.ExportToProj4(). Default to None in which case no reprojection is performed. Data in the database must be stored in 4326 crs
Returns:

A geojson like feature
Return type:

dict
