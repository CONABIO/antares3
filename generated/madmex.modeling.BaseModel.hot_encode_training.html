madmex.modeling.BaseModel.hot_encode_training¶

BaseModel.hot_encode_training(X)[source]¶

Apply one hot encoding to one or several predictors determined by the list of indices of the hot_encode attribute

In case no encoding is required (self.categorical_features is None), simply returns the input array

Parameters:X (array) – The array of predictors
Returns:The array of predictors with specified variables encoded
Return type:array
