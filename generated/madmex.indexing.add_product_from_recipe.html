madmex.indexing.add_product_from_recipe¶

madmex.indexing.add_product_from_recipe(recipe, name=None)[source]¶

Wrapper to add a product from a recipe name

yaml product description file corresponding to the recipe must be present in the ~/.config/madmex/indexing directory.

Parameters:
  • recipe (str) – Name of the recipe
  • name (str) – Product name, optional. Default to None in which case a random product name is generated.
Returns:

List of 2, containing a ProductResource object and a DatasetType object
Return type:

list
