Antares¶

A python library for scalable satellite image analysis, from data dowload to production of national scale maps.

Antares is the name of the CONABIO geopsatial engine. The project started under the name MAD-Mex (Monitoring Activity Data for the Mexican REDD+ program) a few years ago. Back then the objective was to produce national land cover and land cover change information for Mexico (some of the products produced by MAD-Mex can be visualized via the CONABIO geoportal ). MAD-Mex has now evolved into antares, a scalable system that aims to be more generic and flexible geoprocessing engine.

Antares heavily relies and uses open data cube. Other key libraries that make this system possible are xarray, dask and distributed.
